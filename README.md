# 3D Star Keychain 🌟

مشروع تصميم ميدالية مفاتيح ثلاثية الأبعاد على شكل نجمة مجسمة باستخدام برنامج **Onshape**، مجهزة ومعدة بالكامل للطباعة ثلاثية الأبعاد (3D Printing).

---

## 🛠️ الأدوات المستخدمة (Tools)
* **برنامج التصميم:** Onshape (نسخة الآيباد)
* **صيغة الملف المستخرج:** STL 

## 📐 المواصفات القياسية (Specifications)
* **الشكل:** نجمة خماسية الأبعاد متناسقة.
* **ثقب الميدالية:** ثقب علوي مفرغ بدقة بقطر **4 مم** لتمرير حلقة الميدالية بسهولة.

## 🖨️ إرشادات الطباعة ثلاثية الأبعاد (3D Printing Settings)
* **الملف المستخدم:** `star1.stl`
* **وحدة القياس عند الاستيراد:** المليمتر (Millimeter) لضمان المقاسات الصحيحة
